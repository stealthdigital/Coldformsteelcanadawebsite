# Cold Form Steel Canada - Web Application

This project was exported from Figma Make. It is a modern React application built with Vite and Tailwind CSS.

## üöÄ Quick Start

### 1. Prerequisites
- **Node.js** (v18 or higher recommended)
- **Git**

### 2. Install Dependencies
Open your terminal in this folder and run:
```bash
npm install
```

### 3. Run Locally
Start the development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to see the site.

## üåç How to Deploy (Go Live)

We recommend using **GitHub** + **Vercel** for the easiest workflow.

### Step 1: Push to GitHub
1. Create a new empty repository on [GitHub](https://github.com/new).
2. In your project folder terminal, run:
   ```bash
   git init
   git add .
   git commit -m "Initial export from Figma Make"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin main
   ```

### Step 2: Connect to Vercel
1. Go to [Vercel](https://vercel.com).
2. Click **Add New Project**.
3. Import your GitHub repository.
4. Vercel will detect `Vite` automatically. Click **Deploy**.
5. Your site is now live! üöÄ
