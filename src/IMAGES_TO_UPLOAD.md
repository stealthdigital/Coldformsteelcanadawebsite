## GitHub Upload Instructions

### Method 1: Via GitHub Web Interface (Easier for bulk upload)

1. Go to: `https://github.com/stealthdigital/Coldformsteelcanadawebsite`
2. **Switch to `assets` branch** (dropdown at top left)
3. Navigate to `public/assets/` folder
4. Click **"Add file" → "Upload files"**
5. **Drag and drop all image files** at once
6. Add commit message: "Add remaining website images"
7. Click **"Commit changes"**

### Method 2: Via Git Command Line (If you prefer)

```bash
# Clone the assets branch
git clone -b assets https://github.com/stealthdigital/Coldformsteelcanadawebsite.git assets-branch
cd assets-branch/public/assets

# Copy all your downloaded images here
# Then commit and push
git add *.png
git commit -m "Add remaining website images"
git push origin assets
```
