## üìù Implementation Instructions

### For Each Page:

1. **Add HeadMeta import** at the top:
```tsx
import { HeadMeta } from '../HeadMeta';
```

2. **Add HeadMeta component** at the start of the return statement:
```tsx
return (
  <div>
    <HeadMeta 
      title="Your Optimized Title Here"
      description="Your optimized description here"
      image="Image URL here"
    />
    {/* Rest of page content */}
  </div>
);
```

3. **Use title format:**
   - Primary Keyword | Secondary Keyword | Brand Name
   - Keep under 60 characters
   - Include location (Ontario/Canada) where relevant

4. **Use description format:**
   - Start with primary benefit or value proposition
   - Include key features and numbers
   - Include call-to-action or target audience
   - Keep under 155 characters

5. **Image recommendations:**
   - Use actual project photos for Success Stories
   - Use model photos for Piccolo/Medio
   - Use relevant illustrations for articles
   - Size: 1200x630px for optimal social sharing
   - Update placeholder URLs with real images
